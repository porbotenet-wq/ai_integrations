# STSphera — AI Integrations

ERP-система для управления строительными проектами (СПК, алюминий, НВФ).

## Структура

```
gpr_bot/    — Telegram-бот (Python, aiogram, SQLAlchemy, FastAPI)
mini-app/   — Telegram Mini App (Vite + React + TS + Tailwind + Supabase)
```

## Stack

### gpr_bot
- Python 3.11+
- aiogram (Telegram Bot API)
- SQLAlchemy + Alembic (PostgreSQL)
- FastAPI (REST API)
- Docker + docker-compose

### mini-app
- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn-ui
- Supabase (Auth, DB, Edge Functions, Storage)
- PWA (manifest + service worker)

## Визуальный стиль
Architectural Cinema — тёмный бетон, графит, металл, LED-акценты.
Оптимизировано для стройки: крупные тап-зоны (56px+), высокая контрастность.
